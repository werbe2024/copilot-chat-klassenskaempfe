# Band 1: Klassenkämpfe 1914–1945
## Kapitel 11: KI und feministische Kämpfe

---
### 📌 Einleitung
Feministische Theorien zeigen, wie Technologien **Geschlechterungleichheiten** reproduzieren. KI-Systeme wie GitHub Copilot sind oft von **männlichen Entwicklern** dominiert und spiegeln **patriarchale Strukturen** wider. Wie können **feministische Bewegungen** KI demokratisieren?
